FindDesign é um hub centralizado de recursos para designers, reunindo links úteis de ferramentas, bibliotecas, inspirações, bancos de imagens, fontes, mockups e muito mais.
O site oferece busca inteligente, sugestões automáticas, filtros por categoria e uma interface moderna com tema dark.
O projeto foi desenvolvido em HTML, CSS e JavaScript, com foco em acessibilidade, organização e boa experiência do usuário.
